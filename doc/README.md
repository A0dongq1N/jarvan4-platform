## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | Vue3 + TypeScript + ECharts |
| 后端 | Go + trpc-go（http_no_protocol 模式） |
| 压测脚本 | Go 语言（Go plugin 机制动态编译加载），预留 JS 扩展 |
| 持久化 | MySQL |
| 缓存/实时 | Redis |
| 架构模式 | Master-Worker 分布式压测 |
| 实时数据 | 前端 1 秒轮询 |
| 指标上报 | Push（Worker→Master，每秒）+ Pull（Master→Worker，每5秒兜底） |
